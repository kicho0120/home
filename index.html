About Course

본 강의에서는 웹 사이트 제작 기술 중 프론트-엔드(Front-End) 기술로 기본적인 HTML, CSS, Javascript에 대해 배운다. HTML은 웹 페이지를 통해 제공하게 될 컨텐츠 구조화와 작성에 사용되며, CSS는 웹 페이지의 컨텐츠를 좀 더 사용자에게 전달하기 쉽고 이해하기 쉽도록 외양을 꾸미기 위해 용도도 사용되며, Javascript는 웹 문서에서 사용자의 액션에 반응하기 위한 다양한 동적 표현을 위해 사용된다. 이 들 3가지 프론트-엔드 기술은 웹 사이트 구축에 있어 효과적인 클라이언트-측 프로그램 작성을 위해 사용되는 것으로 본 강의에서는 각 요소기술과 이들 요소기술간의 연결에 대해 배우게 된다.
그리고 종합적인 웹 사이트 작성 응용을 위해 Bootstrap 5 Front-end Web Framework을 기반으로 개인별 웹 사이트를 개발한 후 GitHub의 웹 호스팅 서비스를 통해 서비스할 수 있도록 한다.
